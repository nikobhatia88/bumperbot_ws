# BumperBot Workspace

This is my ROS workspace for the BumperBot project.

## Description
Contains the source code, packages, launch files, and configuration for the BumperBot robot, including:
- ROS nodes in Python and C++
- URDF and XACRO robot description files
- Example packages for learning ROS topics, parameters, and publishers/subscribers
- Configuration files for controllers and simulation
